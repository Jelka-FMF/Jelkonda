# Jelkonda

Jelkonda is a system for development of [Jelka](https://jelka.fmf.uni-lj.si/about/) patterns using Python in the browser based development environment. The python source code is run using [Pyodide](https://pyodide.org) runtime. 

# Setup

Run 'setup.sh' to download necessary '.whl' files. This files should be hosted on a static web page server. All other files should also be available.
